## Cloning
1. Clone project git repo
```shell
git clone git@github.com:s20551/house-prediction.git
```
2. Set up your virtual environment
```shell
virtualenv venv
```
3. Install requirements
```shell
pip install -r requirements.txt
```
4. Pull dvc data files
```shell
dvc pull
```
